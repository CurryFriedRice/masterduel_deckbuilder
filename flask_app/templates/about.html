About this application

Hello! My name is Anthony Do and I was the one who built this application

This application was inspired by the release of Yu-Gi-Oh Master Duel. It uses the Yu-Gi-Oh Api by Prodeck.

I understand that there is a lot that can still be done but the purpose of this project was to build out an online deckbuilding experience that you can just boot up and start.

This is not a commercial product and was built for educational purposes all card images belong to Konami Holdings.

Thanks for taking the time to look at this application.

You can check out my github @ Github.com/CurryFriedRice
